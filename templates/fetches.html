If you wanna accumulate some fields, please use address line. For example the query ?homeworld&gender will show you the number of characters with given homeworld and gender. I am not a frontend developer, sorry =)

Also if you wanna download something, just type download after the page number and before question mark(?) in address line. For example the query http://127.0.0.1:8000/characters/3/download?homeworld&gender will download a csv file with given params.
